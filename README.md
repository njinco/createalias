# createalias

Interactive alias manager for bash.

## Usage

Source the script so it can update your current shell session:

```bash
source ./crealias.sh
```

## Download

```bash
curl -fsSL https://raw.githubusercontent.com/njinco/createalias/main/crealias.sh -o crealias.sh
```
